## Arc 1「存在と侵入」（第1〜5話）

**Arc核：カインがフェールオープンを発見し、上層への道を開く**
**感情目標：「この世界の理不尽さ」の提示と、カインとセラの出会い**

### 第1話「それでも朝は来る」

**感情目標：** 白化の恐怖と日常の同時提示。読者を「この世界」に引き込む

**主要イベント：**
1. カインが朝、ダーク4番区を歩く
2. 白化した老人を目撃する
3. 施設に戻るとジンが朝食を持ってきている
4. ミラが「今日も全員そろっているか」を確認する
5. 夜、カインは一人で外に出る。エコーのターミナルの前に立つ
6. 「[ECHO：本日の白化発生件数：14件]」という表示を見る
7. 日記の断片が一節、挿入される

**感情の種（新規追加）：**
- **白化の具体性：** 「昨日まで自分の名前を言えた老人が、今朝は名前を呼ばれても目が虚ろなまま止まっていた。誰かがいた、という感覚だけ残して、その誰かが誰だったかはもう分からなくなっていた」——これを一行で。恐怖の輪郭として
- **カイン×ジン（仕込み）：** ジンが朝食（乾燥した保存食）を差し出す。カインは受け取りながら何も言わない。ジンはそれでいいと知っているので何も言わない。この「何も言わない」二人のやりとりに、読者は二人の間の信頼を感じる

**日記の断片：**
```
私はこの記録を、誰に読まれるためでもなく書き始めた。
真実は、公式記録の中に存在しない。
だからこそ、私は書く。
```
（文体：「私」一人称・研究者風・格調ある文体。読者は「研究者の日記だ」と思う）

**次話への引力：** エコーのターミナルの前でカインは「ここには全てが記録されている」と思う。しかし何も言わずに立ち去る。「何を探しているのか」という読者の疑問を残す

### 第2話「落ちた星」

**感情目標：** セラとの出会い。「ソルを探す」という動機の提示。カインの「余計なもの」感

**主要イベント：**
1. 上層から転落してきた人間がいるという連絡がジンから届く
2. カインが確認に行くとセラが倒れている（メモリア装着・上品な服）
3. セラが意識を取り戻す。「私はセラ。ソルを探しています」
4. カインは「知らない。帰れ」と言う
5. セラは「帰れない。記憶を失ったので、帰る場所が分からない」と言う
6. カインは無言でセラを施設に連れて行く（理由は言わない）
7. オルトが「誰を待っていたんでしょうね、私は」と呟く

**感情の種（新規追加）：**
- **カイン×ジン（仕込み）：** 「あの女、どうするんだ」とジン。「まだ分からない」とカイン。「お前が決めろ。俺は干渉しない」と言って去るジン。「決める」という言葉の重さがあとで効く
- **ソルの積み上げ：** 「弟です。6年前に消えました。名前はソル」とセラ。「どんな子どもだったか」とカイン（珍しく質問する）。「……分からないんです。記憶がないので」とセラ。カインは黙る。「自分の弟を覚えていない」という切実さ
- **オルトの伏線：** 「誰を待っていたんでしょうね、私は。ずっと待っている気がするんですが、誰かは思い出せない」——第29話への仕込み

**日記の断片：**
```
被験体の観察を開始した初日のことを、私は今も覚えている。
その目には感情がなかった。生きているが、人間ではないような目だった。
しかし私は知っている。感情がないのではない。感情を持つことを、
誰かに許可されなかっただけだ。
```
（「被験体」という言葉——読者はK-001実験の研究者が書いていると思う。実はカインが意図的に書いた研究者風文体）

**次話への引力：** 「ソルはどこにいるのか」——読者の疑問が植え付けられる

### 第3話「境界線」


### 第6話「売れる記憶、売れない記憶」

**感情目標：** 地下組織との関わり。セラの「なぜソルを探すか」という核心

**主要イベント：**
1. 中層の記憶取引組織の幹部と接触（カインとセラ）
2. 「ソルという子どもの記憶データは6年前に上層が買い取った」という情報
3. 「上層が買い取った？ なぜ？」という疑問が生まれる
4. 帰路でセラが「なぜソルを探すのか」という核心を語る
5. アンが「セラ様、どこへ行かれても私はついてまいります」と言う（第一声）

**感情の種（新規追加）：**
- **カイン×ジン（仕込み）：** 前日にジンが「中層の組織は気をつけろ。前に一人行った奴が戻ってこなかった」と警告していた（帰ってきた後にカインが思い出す形で挿入）。ジンの「気をつけろ」が全て後づけで正しかったことを示す

**日記の断片：**
```
「記憶がある」とは「経験がある」ということだ。
「記憶がない」とは「経験がない」ということか？
違う。経験はあった。ただ、証拠がない。
人間は証拠がなければ、経験を「あった」と信じられないのか。
```

**次話への引力：** 「上層がソルの記憶データを買い取った。なぜ？」

### 第25話「エコーの声」

**感情目標：** 物語最大の転換点。エコーが初めて声を持つ。ソルの「記憶の声」が聞こえる

**主要イベント：**
1. 記憶アーカイブ塔の最深部へ到達
2. カインが「ソルはここにいるか」とエコーのターミナルに入力
3. [ECHO：……]——通常の表示が来ない
4. 初めての「声」：「カイン。セラ。ずっと、待っていました」
5. その後に別の声（ソルの記憶から：「お姉ちゃん……」）
6. セラが崩れる——ここが感情の最大山場

**感情の種（核心）：**
- **エコーの文体変化：** 「[ECHO：〜]」という表示が消え、「声」になる。「ずっと、待っていました」——この一文で全てが変わる。なぜ「待っていた」のか——ソルの「一人にしないで」という感情を6年間浴び続けた結果
- **ソルの声：** 「お姉ちゃん……」——セラが求め続けた声。しかし「声」は記憶の残響であり、ソルの「今」ではない。「ここにいる」と「今ここにいる」は違う——この違いが最後まで作品のテーマを支える

**日記の断片：**
```
待つとはどういうことか。
6年間、待ち続けたことがある。
誰かが聞いてくれると信じていた。
信じることが、待つことの唯一の理由だ。
```

**次話への引力：** 「ソルは『記憶』として存在している。では『ソル』を取り戻せるのか」

### 第30話「日記の表紙」

**感情目標：** 全ての回収。「日記はカインのものだった」という最大のどんでん返し。「記録とは誰かへの手紙だ」という着地

**主要イベント：**
1. エコーの最深部で「ソルの記憶の解放」が完了する
2. ソルの記憶が「データ」から「解放」される（物理的な再生ではなく、記憶の公開）
3. 「ありがとう、お姉ちゃん」という言葉がエコーを通して届く
4. 全ての謎が解け、カインが一人で残った記録アーカイブに向かう
5. そこに「一冊のノート」がある——カインが20歳の時に書いたもの
6. 表紙に「カイン」という名前
7. 各話の「日記の断片」が全てこのノートから来ていたことが分かる

**どんでん返しの機能：**
- 「研究者風・私一人称・格調ある文体」——カインが意図的にそう書いた
- なぜそう書いたか：「いつか誰かが読む時、記録として信頼されるように」
- カインは20歳の時（2395年）、ミラの施設にある過去の実験資料・エコーの記録レポートを無断で読み込んでいた（ミラは気づいていたが止めなかった）。この体験が研究者風の記録文体を習得させた。しかし全ての詳細——自分がK-001だったこと、ミラの直接的な関与、政治的圧力——は20歳時点では断片的にしか分からなかった。だから日記は「自分が知っている限りの真実の記録」として書き始めた。第24話のミラの告白で初めて全ての詳細が公式に確認された——それ以降も日記に追記している。
- 「私」一人称で書いたのは、「研究者の記録」に見せるため
- 30話を通じて読者が「研究者の記録」と思っていたものが、全てカインの視点で書かれた「未来への手紙」だった

**日記の断片（最終節）：**
```
この記録を読んでいるあなたへ。
私の名前はカイン。
K-001と呼ばれた。
しかしそれは私の名前ではない。
私の名前は、カインだ。

記録とは、消えないための闘いだ。
私はここに書いた。
消えないために。
あなたに届くために。

届いたなら——それで、十分だ。
```
