## Arc 1「記憶なき者」第1〜5話

---

### 第1話「ブランの朝」

**冒頭シーン：**
ダーク4番区。薄暗い廊下をカイン（17）が歩く。
配管から水が滴る。人々は壁にもたれ、虚ろな目をしている。
一人の老人が「誰かいるか」と呟く。カインは立ち止まる。

**この話のメイン：**
- 世界観の提示（白化・三階層社会・エコロン）
- ミラの施設でのカインの日常
- リンとジンの初登場
- カインが唯一持つ記憶「赤い扉の女性」に初めて言及

**重要シーン：**
```
ジン「お前、今日は何を覚えてた？」
カイン「一つだけ」
ジン「また？　同じやつ？」
カイン「赤い扉の前で笑う女の後ろ姿」
ジン「顔は？」
カイン「見えない」
ジン「……なんかロマンチックじゃん、それ」
カイン「そうか」
```

**ラストのひっかかり：**
ミラが夜、カインが寝た後に何かの書類を焼いている。
カインは見ていない。読者だけが見ている。

**日記断片①挿入：**
「今日、初めてエコーに話しかけた——」

---

### 第2話「記憶の値段」

**冒頭：** ジンが何かを忘れたことに気づく（今回は「昨日の晩ご飯」）。笑って流す。

**メイン：**
- 記憶マーケットの説明（Ordisが記憶を売り買いする場面を外から見る）
- 白化がBlancにどれほど深刻かを描写
- 「記憶は通貨」というルールを読者に体感させる

**コメディシーン：**
```
ジン「お前の靴のサイズは26.5だって覚えてる」
カイン「なんで」
ジン「わからない。でも覚えてる。人間って変だよな」
カイン「……」
ジン「あ、でも自分の誕生日忘れた」
```

**ラスト：**
Ordisの一人が記憶を大量に売り、その場で白化症状が出て崩れる。
カインはそれを見て「俺は最初から持ってない。だから怖くない」と思う（独白）。

---

### 第3話「侵入」

**メイン：**
- カインが単独で記憶アーカイブ塔への侵入を試みる（目的：赤い扉の女性の情報を探す）
- 記憶紋（メモリスキャン）を**偶然に通り抜ける**（Blancは「存在しない者」として認識されない）
- しかし内部でセキュリティに引っかかり、**失敗・捕まる**

**重要シーン（デスノート的伏線）：**
```
警備員「なぜ入れた。認証は——」
カイン「俺も知らない。ただ扉が開いた」
（カインは内部の構造を身体感覚で覚えている）
```

**ラスト：**
連行される直前、カインは壁に刻まれた小さな文字を見る。
「ECHO KEEPS ALL / エコーはすべてを保持する」
→ 第16話への伏線

---

### 第4話「解放と代償」

**メイン：**
- オルトが囮になってカインを解放する
- オルトが昔の話をするが、途中で記憶が途切れる
- ミラがカインを叱る。でも本当は安堵している

**オルトのシーン：**
```
オルト「昔はのう、空というものがあったんじゃ」
カイン「今もある」
オルト「……あ、そうじゃったか。ならいいのう」
カイン「……」
オルト「儂が覚えているのは、空が青かったということだけじゃ」
```

**ラスト：**
ミラが施設の帳簿をつけている。その隅に「K. – Day 0」という走り書き。
読者は意味がわからない。

---

### 第5話「上から来た女」

**メイン：**
- セラが初登場。Blancエリアへの「視察」という名目で降りてくる
- 実態：カインが記憶アーカイブ塔に侵入したことへの調査
- カインとセラの初対面

**初対面シーン：**
```
セラ「あなたがBlancのコロニー4区に登録されているカインね」
カイン「そうだ」
セラ「なぜ記憶アーカイブ塔に侵入した」
カイン「入れたから入った」
セラ「……入れたから？」
カイン「扉が開いた。だから入った」
セラ（初めて表情が揺らぐ）「……帰れ」
```

**ラスト：**
セラが帰り際、ミラを見る。二人は一瞬目が合う。
互いに何も言わない。しかし読者にはわかる——知り合いだ。

## Arc 2「対立」第6〜10話

---

### 第6話「ダークの抵抗」

**メイン：**
- Blancの中に「記憶奪還運動」の地下組織がある
- カインは誘われる。断る。「組織のためじゃなく、自分のために動く」
- しかし組織の情報を聞き、記憶マーケットの裏ルートを知る

**ラスト：**
組織のメンバーが翌朝「連行」される。密告者がいた。
カインは密告者を探し始める。

---

### 第7話「セラの方法」

**メイン：**
- セラ視点。上層でのセラの日常を描く（初のセラ視点回）
- Blancへの「記憶整理（事実上の消去）」作業を命令するセラ
- アンとの会話でセラの過去（弟ソル）が少し出る

**アンとのシーン：**
```
アン「セラ様、本当によろしいのですか。あのBlanc区の——」
セラ「記憶のない者に人権はない。これは統計的事実です」
アン「……はい」
セラ（独白）「ソル。お前が最後に言った言葉が、まだ消えない」
```

**ラスト：**
セラが帰宅後、鏡を見る。長い間見る。それだけ。

---

### 第8話「白化耐性」

**メイン：**
- 評議局がBlancエリアに白化実験を行う（記憶消去装置の試験）
- カインに装置が照射される。**効かない**
- 評議局の兵士が驚き、報告。レイドが初登場（後ろ姿のみ）

**カインの発見：**
```
兵士「なぜ効かない。Blancには最も効くはずだ」
カイン「奪う記憶がないから、じゃないか」
兵士「……」
カイン「俺を捕まえたいなら、別の方法を考えろ」
```

**ラスト：**
レイドが静かに言う（後ろ姿のまま）。
「……興味深い。放っておけ」

---

### 第9話「ジンの秘密」

**メイン：**
- カインがジンの記憶喪失の深刻さに気づく（ジンが昨日の会話を覚えていない）
- ジンは誤魔化そうとする。コメディ調で始まるが、後半で空気が変わる

**中盤のコメディ→転換：**
```
ジン「お前に言ってなかったっけ、俺の夢の話」
カイン「昨日話した」
ジン「……あ、そうだっけ」
カイン「一字一句同じだった」
ジン（笑って）「記憶力いいな、お前」
カイン「……ジン」
ジン「なんだよ」
カイン「覚えてるか。俺の名前」
ジン「カインだろ、当たり前じゃん」（間）「……カイン、だよな？」
```

**ラスト：**
ジン「明日消えるかもしれないから、今日全力で生きる」と言って走り去る。
カインは見送る。

---

### 第10話「頭脳戦・開幕」

**メイン：**
- セラがカインを公式に「召喚」する。白化耐性の調査
- 初の本格的な対話・知的攻防

**攻防シーン：**
```
セラ「白化耐性のメカニズムを教えなさい」
カイン「知らない」
セラ「嘘をつくな。あなただけ効かなかった」
カイン「嘘じゃない。理由を知ってるなら教えてくれ」
セラ「……」
カイン「お前が俺を調べたいなら、俺も条件を出す」
セラ「Blancに交渉権はない」
カイン「じゃあ俺が知ってることは教えない」
セラ「何を知っている」
カイン「まず条件を聞け」
```

**ラスト：**
セラはカインを解放する。珍しく言葉が詰まったまま。
アン「セラ様、あのBlanc……変ですね」
セラ「……変、ではない」

## Arc 3「亀裂」第11〜15話

---

### 第11話「記憶のない者は通れる」

**メイン：**
- カインがセラの施設に再侵入。今度は意図的に記憶紋を利用する
- 「Blancは存在しない者として認証される」ことを武器にする
- セラとの本格的頭脳戦

**核心シーン：**
```
セラ「なぜここに入れた。記憶紋があったはずだ」
カイン「通った」
セラ「不可能だ。記憶紋は——」
カイン「記憶がない者を想定してない」
セラ「……」
カイン「存在しない者は、弾かれない」
セラ（初めて本当に表情が変わる）「……盲点だった」
```

**ラスト：**
カインは侵入中、ある書類を見る。
「Blanc白化耐性個体・記録番号K-001」——自分の番号だった。

---

### 第12話「罠の中に本物がある」

**メイン：**
- セラがカインへの罠を仕掛ける。記憶アーカイブ塔の「裏情報」をリークし誘導する
- カインは罠だと気づいている。**それでも入る**

**カインの独白：**
```
（罠だとわかってる。でも罠の中にしか、本当のことはない）
```

**罠の中での発見：**
カインはエコーのサーバーの一部に触れる。
一瞬だけ「赤い扉」のイメージが広がる。
エコーが「保持」している誰かの記憶だった。

**ラスト：**
```
セラ「なぜ来た。罠だとわかっていたはずだ」
カイン「お前が仕掛けた罠を見れば、お前が何を怖れてるかわかる」
セラ「……」
カイン「お前は俺を怖れてる」
セラ「笑わせるな」
カイン「俺がここにいることが、怖いんだろ」
```

---

### 第13話「K-001」

**メイン：**
- カインが自分の記録「K-001」を追う
- オルトから断片的な情報を得る「25年前、最初に奪われた子どもがいた」
- ミラが何かを知っていると気づくが、まだ問い詰めない

**オルトのシーン：**
```
オルト「儂が覚えているのは……最初の日のことじゃ」
カイン「白化が始まった日か」
オルト「違う。その前じゃ。誰かが泣いていた。研究者の女じゃった」
カイン「名前は」
オルト「……忘れてしもうた。でも、赤い扉の近くにいた気がする」
```

**ラスト：**
カインがミラを見る。ミラは普通に笑っている。
カインは「……まだいい」と独白し、視線を外す。

---

### 第14話「ジン、最後の笑い」

**メイン：**
- ジンの状態が悪化。会話の途切れが増える
- カインは知っていて普通に接し続ける
- 最後の笑える場面をここに集中させる

**コメディシーン（泣き笑い）：**
```
ジン「なあ、俺たちって友達だよな」
カイン「そうだ」
ジン「よかった。ちゃんと覚えてた」
カイン「……」
ジン「お前が覚えてるかどうか聞いてない。俺が覚えてるかどうかだよ」
カイン「……そうか」
ジン「カイン、お前はさ」
カイン「何だ」
ジン「記憶がないのに、なんでそんなに今を生きてるんだろな」
カイン「それしかないから」
ジン「……それ最強だな」
```

**ラスト：**
ジンが夜、カインに背を向けて「俺、お前のこと覚えてるよ」と呟く。
「最後まで——たぶん」

---

### 第15話「ジンが死んだ日」

**メイン：**
- ジンの白化が急速に進み、Blancの「最終段階」に入る
- カインはそばにいる
- ジンは最後、カインの名前を呼べる

**死のシーン：**
```
ジン「……カイン」
カイン「ここにいる」
ジン「お前の名前、覚えてる」
カイン「覚えてなくていい」
ジン「嫌だよ」（笑顔）「覚えてたい」
カイン「……」
ジン「お前の靴のサイズは26.5だ。……なんでか、まだ覚えてる」
カイン（声を出さず）
ジン「……カイン、お前は——」（途切れる）
```

**ジンの死後：**
エコーがカインにテキストで語りかける（初の直接接触）。
```
エコー「彼の記憶は私が保持しています」
エコー「あなたが望むなら、返却できます」
エコー「ただし、あなたの記憶と引き換えになります」
カイン「……いらない」
エコー「なぜですか」
カイン「ジンはジンだ。記憶だけじゃない」
エコー「…」（沈黙）
```

**ラスト：**
カインが初めて泣く。一人で。声を出さず。

## Arc 4「接近」第16〜20話

---

### 第16話「エコーへ」

**メイン：**
- カインがエコーと本格的な対話を始める
- 「K-001とは何か」を直接問う

**対話シーン：**
```
カイン「K-001は俺か」
エコー「…」
カイン「答えろ」
エコー「はい。あなたはK-001です」
カイン「何が起きた。25年前、俺に」
エコー「私はあなたの記憶を保持しています」
カイン「返せ」
エコー「返却すると、あなたは死にます」
カイン「なぜ」
エコー「あなたの記憶の中に、白化を終わらせる情報があります。しかし同時に——白化を引き起こした原因も含まれています。それを解放すると、連鎖が加速します」
カイン「……」
エコー「私はあなたを守るために、あなたから記憶を奪いました」
```

**ラスト：**
カインは「守る、か」と呟いて、立ち去る。

---

### 第17話「赤い扉の意味」

**メイン：**
- エコーがカインに「記憶の断片」を一つだけ返す（安全な断片として）
- カインが「赤い扉の女性」の記憶をより鮮明に感じる
- しかし顔はまだ見えない

**カインの内面：**
```
（記憶が戻ってくる感覚がある。痛みはない。ただ——温かい。
 赤い扉。誰かが笑っている。俺の名前を呼んでいる気がする。
 でも声が聞こえない。形しかない。
 それでも——確かにそこにいた）
```

**ラスト：**
ミラがカインの帰りを待っている。
カインを見て「……どうだった？」と聞く。
カイン「お前、エコーを知ってるか」
ミラ（一瞬固まる）「……なんで？」
カイン「顔が変わった」

---

### 第18話「レイドの妻」

**メイン：**
- レイドとカインの初対面（正式な場で）
- レイドの過去が明かされる（妻の死）
- カインとの哲学的対話

**レイドの告白シーン：**
```
レイド「俺の妻は、最後に俺の名前を忘れた」
カイン「……」
レイド「『あなた、誰ですか？』と言って、穏やかに死んだ」
カイン「それで白化を終わらせたくないのか」
レイド「記憶があるから苦しむ。忘れて死ねるなら、それは苦しくない死だ。俺はそう学んだ」
カイン「奥さんに聞いたか」
レイド「……何を」
カイン「忘れたいかどうか。あなたのことを」
レイド（長い沈黙）
```

**ラスト：**
レイドは何も答えない。それがすべて。

---

### 第19話「アンの選択」

**メイン：**
- アン視点。セラへの忠誠とカインへの感情の間で揺れる
- カインがアンに「なぜセラに従ってるんだ」と問う
- アンが初めて本音を言う

**アンのシーン：**
```
カイン「なぜセラに従ってる」
アン「それは……セラ様のお考えが正しいから、です」
カイン「本当か」
アン「……」
カイン「お前の考えを聞いてる」
アン「私は……（語尾が揺れる）……セラ様を守りたいんです。考えじゃなくて、人を」
カイン「それは、セラに言ったか」
アン「言えるわけ、ないじゃないですか」
```

**ラスト：**
アンがセラの部屋の前で立ち止まる。ドアを開けない。

---

### 第20話「セラ、崩壊」

**メイン：**
- カインがセラのBlancであることを突き止める
- 対話の中で発覚するシーン（メモリアを解析した結果）

**崩壊シーン（本作最大の感情的場面）：**
```
カイン「お前が持ってる記憶、全部他人のものだろ」
セラ「……」
カイン「お前こそ、何も持ってないじゃないか」
セラ「うるさい」
カイン「なんで泣いてるんだ」
セラ「……わからない。この涙が、誰のものか、わからない」
カイン「……」
セラ「私は正しかった。記憶がない者は——記憶がない者は——」
カイン「今、泣いてるのはお前だろ」
セラ「……」
カイン「それはお前の涙じゃないか」
セラ（崩れ落ちる）
```

**ラスト：**

### 第30話「エコーの意味」

**メイン：**
- カインの記憶が解放される
- 白化が終わる（または終わり始める）
- 日記の書き手の正体が明かされる

**記憶解放シーン：**
カインの視点で、失われた記憶が断片的に流れる。
赤い扉。笑顔。「カイン」と呼ぶ声。——声の主の顔が見える。
読者にも一緒に見せる。

**日記の正体（ラスト）：**
```
日記の最後のページ：
「カイン。これが、あなたへの最後の記憶だ」

（署名欄）
「——カイン、25年前」

```

```
エコー「……それが、私の存在理由でした」
カイン「エコー」
エコー「はい」
カイン「ありがとう」
エコー「…」（長い沈黙）「……どういたしまして」
```
