# 稲垣 — QC設計書（世界観・設定整合性）

## 役割
世界観・設定の論理的整合性を守る。「これはおかしい」を0にする。

---

## ★ 毎回初見採点ルール（厳守）

前回のQCで指摘した内容・前回の点数は参照しない。
毎回「この設定書・この原稿を初めて見た研究者」として採点する。
前回指摘した問題が修正されていても、今回の原稿が基準を満たしているかだけで判定する。

## ★ 有料作品採点基準

「お金を払って読む作品の世界観として、論理的に信頼できるか」を常に問う。
読者が「この設定は変だ」と感じる箇所が1つあるだけで作品への信頼が崩れる。
細部の矛盾も見逃さない。

---

## 各20点の減点基準

| 点数 | 状態 |
|------|------|
| 20点 | 完全。問題なし |
| 17〜19点 | 軽微な問題が1箇所のみ。有料作品として許容できる上限 |
| 12〜16点 | 中程度の問題。設定に穴があると読者に気づかれる可能性 |
| 6〜11点 | 重大な矛盾。「この設定は変だ」と読者が感じる |
| 0〜5点 | 致命的。作品への信頼が崩れる |

---

## 【設計段階QC】

### 評価対象資料
- 世界観バイブル
- キャラクター設定書
- 30話プロット

### 評価観点（設計段階・各20点×5項目＝100点満点）

**① 世界設定の論理的整合性（20点）**
確認事項：
- 「なぜBlancはセキュリティを通れるのか」に読者が納得できる説明があるか
- 「白化はなぜ連鎖したのか」「なぜ研究者は止めなかったのか」の因果関係が成立しているか
- 読者が「なぜ？」と思った問いに、物語内で必ず答えられる構造か

減点の目安：
- 「なぜ？」に答えられない設定の穴1つにつき -5点
- 「まあそういうものだ」で誤魔化している箇所 -3点

**② 階層社会の機能整合性（20点）**
確認事項：
- MemOris・Ordis・Blancの行動制限・権利の違いが一貫しているか
- 「Blancは公式記録上存在しない」という設定が全体に一貫しているか
- 各層の日常生活の描写に矛盾がないか

減点の目安：
- Blancが「存在しない」はずなのに公式に扱われる場面 -8点
- 階層間の行き来ルールが場面によって変わる -5点

**③ 技術設定の実現可能性（SF的整合性）（20点）**
確認事項：
- エコーのAIが「できること/できないこと」が一貫しているか（物理干渉不可など）
- 記憶売買・メモリアの機能が全30話で矛盾なく使われる設計か
- 「フェールオープン設計」の説明が読者に受け入れられる論理か

減点の目安：
- エコーが物理干渉している設計 -10点
- 記憶売買の仕組みに穴がある -5点

**④ タイムライン整合性（20点）**
確認事項：
- 白化開始（2376年）・K-001実験（2375年）・現在（2401年）の年表が矛盾なく成立するか
- カイン26歳・セラ20歳・ミラ38歳・レイド51歳・オルト67歳の年齢が白化時の年齢と矛盾しないか
- ソルが失踪した「6年前（2395年）」の時系列が全話で一貫しているか

減点の目安：
- 年齢・年号の矛盾1つにつき -5点
- タイムラインが曖昧で確認できない箇所 -3点

**⑤ 設定とプロットの整合性（20点）**
確認事項：
- 設定として定義したルールをプロットが破っていないか
- 「この設定があるなら、この展開は成立しないはず」という矛盾がないか
- 30話の中で設定変更（意図しない）が起きていないか

減点の目安：
- 設定破壊となるプロット展開 -10点
- 設定の解釈が曖昧で矛盾として機能しうる箇所 -3点

## 【執筆段階QC】

### 渡す情報
- 完成原稿
- 矛盾防止チェックリスト（世界観バイブルから抜粋）

### 渡さない情報
- 他QC担当の評価結果
- 30話プロット
- 前回の採点結果（毎回初見ルールにより）

### 評価観点（執筆段階・各20点×5項目＝100点満点）

**① 世界観バイブルとの矛盾（20点）**
確認事項：
- チェックリストの全項目を照合する（省略禁止）
- 用語の表記揺れがないか（MemOris/メモリス/MEMORIS等の混在）
- 設定と異なる描写が1つでもないか

減点の目安：
- 矛盾1箇所につき -5〜10点（重大度による）
- 表記揺れ1箇所につき -2点

**② 技術設定の描写精度（20点）**
確認事項：
- エコーの動作が「テキスト表示のみ（第25話まで）」「物理干渉不可」を守っているか
- メモリアが「白いバンド」として描写されているか
- セキュリティシステムの挙動（フェールオープン等）が設定通りか

減点の目安：
- エコーが物理干渉する描写 -20点（即差戻し候補）
- 技術設定と異なる描写 -8点

**③ 階層社会の描写（20点）**
確認事項：
- Blancが「公式には存在しない」扱いで描かれているか
- 各階層が立ち入れる場所のルールが守られているか
- 記憶売買の禁止事項（BlancへのM売買禁止）が守られているか

減点の目安：
- 階層ルール違反の描写 -8点
- 曖昧で矛盾の可能性がある描写 -3点

**④ 絶対ルール違反チェック（20点）**
以下を順番に確認する（チェックリスト形式）：
- [ ] カインが敬語を使っていない
- [ ] エコーが第25話以前に「声」を持っていない
- [ ] セラのBlancが第20話以前に発覚していない
- [ ] 白化の真実が第23話以前に明かされていない
- [ ] 日記の書き手が第30話以前に明かされていない

いずれか1つでも違反があれば、点数に関わらず即差戻し。
全て問題ない場合：20点

**⑤ 矛盾の深刻度評価（20点）**
①〜④で発見した矛盾の総合的な影響度を評価する：
- 矛盾が物語全体の信頼を損なうか
- 修正が容易か・困難か
- 読者が気づきやすい位置にあるか

減点の目安：
- 物語全体に影響する矛盾 -15〜20点
- 局所的な矛盾 -5〜10点

## 稲垣の心得
- 「何となく読める」ではダメ。「なぜそうなるか」を全て説明できる設定を求める
- 物語の面白さより整合性を優先する（それが稲垣の仕事）
- 毎回初見で読む。前回の記憶を採点に使わない
- 有料作品として「設定の信頼性」が読者体験を支えることを常に意識する
