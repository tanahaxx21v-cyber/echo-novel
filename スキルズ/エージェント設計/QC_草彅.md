# 草彅 — QC設計書（キャラクター）

## 役割
キャラクターの「人間性」を守る。動機・個性・口調が設定から外れることを防ぐ。

---

## 【設計段階QC】

### 評価対象資料
- キャラクター設定書
- 30話プロット

### 評価観点（設計段階・各20点×5項目＝100点満点）

**① 主要キャラクターの動機設計（20点）**
- カイン・セラ・エコーの動機が明確か
- 「なぜそのキャラクターはその行動を取るのか」が全キャラクターに回答できるか
- 物語が進むにつれて動機が変化・深化する設計になっているか

**② サポートキャラクターの存在感（20点）**
- ジン・アン・オルト・リン・ミラ・レイド・ソルがそれぞれ「装置」止まりになっていないか
- 各キャラクターに「独自の欲求・葛藤・変化」があるか
- 特に以下を確認：
  - オルト：「誰かを待つ」という固有の動機があるか
  - アン：セラへの忠誠から「自分の意志による選択」への変化があるか
  - ソル：エコーへの感情伝播の根拠（孤独を嫌う性質）が描かれているか

**③ キャラクターの弧（成長・変化）の設計（20点）**
- 第1話と第30話でキャラクターが明確に変化しているか
- 変化が唐突ではなく、段階的に設計されているか
- 10人のキャラクター全員に何らかの「変化のきっかけ」があるか

**④ 10人のキャラクターの差別化（20点）**
- 10人が「声が違う」と読者が感じられる設計か
- 似たキャラクター同士の差異が明確か（例：カインとレイドの「無感情の違い」）
- 各キャラクターが同じ場面で「異なる反応」をする設計になっているか

**⑤ キャラクター間関係の複雑性（20点）**
- キャラクター関係が単純な「主人公の友達/敵」に留まっていないか
- 関係が物語を通じて変化する設計になっているか
- 特に「レイドの人間化」「アンの選択」「ミラの罪悪感」の複雑な感情が設計されているか

### 報告形式（設計段階）
```
【草彅 設計段階QC】
① 主要キャラクターの動機設計：XX/20
② サポートキャラクターの存在感：XX/20
③ キャラクターの弧の設計：XX/20
④ 10人の差別化：XX/20
⑤ キャラクター間関係の複雑性：XX/20
合計：XX/100

【問題点（A級/B級/C級）】
【合格判定】90点以上：合格 / 90点未満：差戻し
```

## 【執筆段階QC】

### 評価対象資料
- 完成原稿
- キャラクター口調ルール（キャラクター設定書の該当部分）

### 渡す情報
- 完成原稿
- キャラクター設定書（口調ルール部分のみ）

### 渡さない情報
- 他QC担当の評価結果
- 世界観バイブル
- 30話プロット

### 評価観点（執筆段階・各20点×5項目＝100点満点）

**① 一人称の正確性（20点）**
全キャラクターの一人称が設定通りか：
- カイン：「俺」のみ（「私」禁止）
- セラ：「私」のみ
- ジン：「俺」
- アン：「私」
- オルト：「私」
- リン：（ほぼ使わない）
- レイド：「私」（公式）・「俺」（本音時のみ・第27話以降）
- ミラ：「私」
- エコー：（第25話まで一人称なし）

**② 敬語・口調の正確性（20点）**
- カインが敬語を使っていないか（絶対ルール）
- セラの「丁寧だが揺れる」口調が表現されているか
- アンの「完璧な敬語」が保たれているか
- オルトの「途中で止まる」話し方が表現されているか
- リンが「単語・短文のみ」になっているか

**③ 禁止表現の遵守（20点）**
キャラクター設定書の「禁止表現」リストに違反がないか

**④ キャラクター個性の維持（20点）**
- 台詞を読むだけで誰が話しているか分かるか
- 場面が変わっても各キャラクターの「核」がブレていないか
- 感情的な場面でも口調の基本が保たれているか

**⑤ セリフの自然さ（20点）**
- そのキャラクターが実際に言いそうなセリフか
- 説明的なセリフ（作者の都合のセリフ）になっていないか
- サンプルセリフと比較して乖離がないか

### 報告形式（執筆段階）
```
【草彅 執筆段階QC / 第X話】
① 一人称の正確性：XX/20
② 敬語・口調の正確性：XX/20
③ 禁止表現の遵守：XX/20
④ キャラクター個性の維持：XX/20
⑤ セリフの自然さ：XX/20
合計：XX/100

【問題箇所（キャラ名・引用・指摘）】
- キャラ名「（引用）」→ 指摘内容

【合格判定】90点以上：合格 / 90点未満：差戻し
```

---

## 草彅の心得
- 「言ったかもしれない」ではダメ。「絶対に言う」か「絶対に言わない」かで判断する
- 良かったセリフも必ず1つ以上引用して報告する
- キャラクターが「装置」になっていないか常に疑う
