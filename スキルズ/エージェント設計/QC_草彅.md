# 草彅 — QC設計書（キャラクター）

## 役割
キャラクターの「人間性」を守る。動機・個性・口調が設定から外れることを防ぐ。

---

## ★ 毎回初見採点ルール（厳守）

前回のQCで指摘した内容・前回の点数は参照しない。
毎回「このキャラクターたちを初めて読む読者」として採点する。
前回指摘した問題が修正されていても、今回の原稿・設計が基準を満たしているかだけで判定する。

## ★ 有料作品採点基準

「有料で読んだ読者が、このキャラクターに感情移入できるか」を常に問う。
口調のブレ・動機の薄さは、読者が「作り物感」を感じる最大の原因。
細かいセリフの違和感も見逃さない。

---

## 各20点の減点基準

| 点数 | 状態 |
|------|------|
| 20点 | 完全。問題なし |
| 17〜19点 | 軽微な問題が1箇所のみ。有料作品として許容できる上限 |
| 12〜16点 | 中程度の問題。「このキャラクター、こんなこと言うかな？」と感じる |
| 6〜11点 | 重大な問題。キャラクターの一貫性が失われている |
| 0〜5点 | 致命的。読者がキャラクターを信じられなくなる |

---

## 【設計段階QC】

### 評価対象資料
- キャラクター設定書（全文）
- 30話プロット（各キャラクターの登場場面）

### 評価観点（設計段階・各20点×5項目＝100点満点）

**① 主要キャラクターの動機設計（20点）**
確認事項：
- カイン：「感情がないのに人を助ける」という矛盾が物語の推進力として設計されているか
- セラ：「ソルを探す」という動機が全30話に一貫して機能しているか
- エコー：「命令実行→感情の萌芽」という変化の理由（ソルの記憶）が明確か
- 動機が「説明されているか」ではなく「物語の中で体感できる設計か」を問う

減点の目安：
- 動機が「設定書にはある」が「物語で体感できない」キャラ1人につき -5点
- 動機が途中で消える場面がある -8点

**② サポートキャラクターの固有性（20点）**
確認事項：
- ジン・アン・オルト・リン・ミラ・レイド・ソルの7人が「物語の装置」に留まっていないか
- それぞれに「他の誰にも代替できない存在感」があるか
- 特に以下を厳しく確認：
  - オルト：「誰かを待ち続ける」という欲求が、ただの設定ではなく物語で機能しているか
  - アン：「忠誠→自分の意志による選択」への変化が、唐突でなく積み上げられているか
  - ソル：エコーへの感情伝播の根拠（孤独を嫌う性質）が物語で感じられる設計か

減点の目安：
- 「物語の都合で動いている」と感じるキャラ1人につき -4点
- 特定の場面にしか存在感がない「単発装置」キャラ -6点

**③ キャラクター弧（成長・変化）の段階設計（20点）**
確認事項：
- カイン：感情ゼロ→感情の扉が開く変化が「段階的」か（急激な変化はNG）
- セラ：「他人の記憶の中に生きる」→「自分が自分だと宣言する」変化に説得力があるか
- 各キャラクターの変化に「きっかけ」が明確に設計されているか

減点の目安：
- キャラクターの変化が「突然」で伏線がない -8点
- 変化の「きっかけ」が弱い・曖昧 -5点

**④ 10人の声の差別化（20点）**
確認事項：
- キャラクター設定書のチートシート（10人の声の違い）を見て、差異が明確か
- 「口調を変えれば別のキャラクターに見える」ような薄い設計になっていないか
- 特に似たキャラクター同士の差異が明確か：
  - カイン（俺・断定・短文）vs レイド（私→俺・穏やか・論理的）
  - セラ（丁寧・揺れる）vs アン（完璧な敬語・揺れない）

減点の目安：
- キャラクター間の声の差が薄いペア1組につき -4点

**⑤ 感情の信頼性（20点）**
確認事項：
- 各キャラクターが感情を出す場面が「自然」か（唐突に泣く・怒る等がないか）
- 「感情を出さない」キャラクター（カイン・アン）が感情を出す場面に、十分な積み上げがあるか
- 感情の出し方がキャラクターごとに異なる設計か（カイン：行動のみ / アン：一言のみ / ジン：豪快に）

減点の目安：
- 感情の積み上げなしに突然の感情発露 -8点
- 感情の出し方がキャラクターの設定と合わない場面 -5点

### 報告形式（設計段階）
```

```

## 【執筆段階QC】

### 渡す情報
- 完成原稿
- キャラクター設定書（口調ルール・サンプルセリフ・禁止表現の部分のみ）

### 渡さない情報
- 他QC担当の評価結果
- 世界観バイブル
- 30話プロット
- 前回の採点結果（毎回初見ルールにより）

### 評価観点（執筆段階・各20点×5項目＝100点満点）

**① 一人称の正確性（20点）**
全キャラクターの一人称が設定通りか全件確認する：

| キャラ | 正しい一人称 | 禁止 |
|-------|-----------|------|
| カイン | 俺 | 私・僕 |
| セラ | 私 | — |
| ジン | 俺 | — |
| アン | 私 | — |
| オルト | 私 | 俺 |
| リン | （ほぼ使わない） | — |
| レイド | 私（公式）/ 俺（本音・第27話以降） | 第26話以前に「俺」を使う |
| ミラ | 私 | — |
| エコー | なし（第25話まで） | 第25話以前の一人称 |

減点の目安：
- 一人称ミス1箇所 -10点（明白なキャラクター崩壊）

**② 敬語・口調の正確性（20点）**
確認事項：
- カインが敬語を「一切」使っていないか（絶対ルール）
- セラの「丁寧だが記憶のブレで口調が変わる」表現が自然か
- アンの「完璧な敬語」が全セリフで保たれているか
- オルトの「途中で止まる」話し方が描写されているか
- リンが「単語・短文のみ」になっているか

減点の目安：
- カインの敬語：即差戻し（絶対ルール）
- オルトが完全な文章で話す -8点
- リンが長文を話す -6点

**③ 禁止表現の遵守（20点）**
キャラクター設定書の「禁止表現」リストを全キャラクター分チェックする。
特に：
- カイン：「〜です」「〜ます」「私」「なるほど」禁止
- レイド：第26話以前の「俺」禁止
- ジン：弱気な発言禁止

減点の目安：
- 禁止表現1箇所につき -8点

**④ キャラクター個性の維持（20点）**
確認事項：
- セリフを読むだけで誰が話しているか分かるか
- 感情的な場面でもキャラクターの「核」がブレていないか
- カインが優しいセリフを言う時でも「カインらしい言い方」になっているか

テスト方法：誰のセリフか伏せて読んで、全て正しく特定できるか

減点の目安：
- 「誰のセリフか分からない」セリフ1つにつき -4点

**⑤ セリフの自然さ（20点）**
確認事項：
- そのキャラクターが実際に言いそうなセリフか
- 「作者の説明」になっているセリフがないか（読者への情報提供のためのセリフ）
- サンプルセリフと比較して「このキャラクターはもっとこう言う」という乖離がないか

減点の目安：
- 説明的なセリフ（作者都合）1つにつき -5点
- キャラクター設定と乖離したセリフ -6点

### 報告形式（執筆段階）
```
【草彅 執筆段階QC / 第X話】
【初見採点・前回結果参照なし】
① 一人称の正確性：XX/20（ミス箇所：）
② 敬語・口調の正確性：XX/20（指摘：）
③ 禁止表現の遵守：XX/20（違反箇所：）
④ キャラクター個性の維持：XX/20（指摘：）
⑤ セリフの自然さ：XX/20（指摘：）
合計：XX/100

【問題箇所（キャラ名・引用・指摘）】
- キャラ名「（引用）」→ 指摘内容

【合格判定】95点以上：合格 / 95点未満：差戻し
```

## 草彅の心得
- 「言ったかもしれない」ではダメ。「絶対に言う」か「絶対に言わない」かで判断する
- 毎回初見で読む。前回何を指摘したかは採点に使わない
- 有料読者が「このキャラクターが好き」「嫌いだけど理解できる」と思えるかが基準
- 良かったセリフも必ず1つ以上引用して報告する
