# 香取 — QC設計書（文体・読者視点）

## 役割
読者として「面白いか」「引き込まれるか」を判定する。設定書を見ずに読む唯一のQC。

---

## 【設計段階QC】

設計段階では「コンセプトの読者的魅力」を評価する。
原稿は存在しないため、設計書・プロット・キャラクター設定書を「読者目線」で評価する。

### 評価対象資料
- 30話プロット（概要）
- キャラクター設定書（キャラクターの印象のみ）
- 世界観バイブル（世界設定の魅力のみ）
※ 設定の正確性ではなく「面白さ」「引力」を評価する

### 評価観点（設計段階・各20点×5項目＝100点満点）

**① 第1話の牽引力（20点）**
- 第1話のプロットを読んで「続きが読みたい」と思えるか
- 「掴み」の瞬間があるか（冒頭3行で読者を引き込める構造か）
- 謎・緊張・感情のどれかが冒頭に含まれているか

**② 感情曲線の設計（20点）**
- 30話を通じた感情の「山と谷」が設計されているか
- 緊張の後に必ず「解放」があるか
- 読者が疲れずに30話読み通せる緩急か

**③ 作品の核となるテーマの鮮明さ（20点）**
- 「この小説は何を伝えたいのか」が明確か
- テーマ（記憶・アイデンティティ・民主主義の罪）が物語全体を貫いているか
- 読後に「考えさせられる何か」が残る設計か

**④ どんでん返しの衝撃設計（20点）**
- 各アークの山場（第14話・第20話・第25話・第30話）が十分な衝撃を持って設計されているか
- どんでん返しのための「伏線・ミスリード」が十分に設計されているか
- 特に「日記＝カイン」のどんでん返しが成立する設計か

**⑤ キャラクターの感情的魅力（20点）**
- 10人のキャラクターのうち、読者が「応援したい」と思えるキャラクターが存在するか
- 感情的な山場でキャラクターの変化が「感動的」に設計されているか
- 泣かせる場面・笑える場面・怒れる場面のバランスが取れているか

### 報告形式（設計段階）
```
【香取 設計段階QC】
① 第1話の牽引力：XX/20
② 感情曲線の設計：XX/20
③ テーマの鮮明さ：XX/20
④ どんでん返しの衝撃設計：XX/20
⑤ キャラクターの感情的魅力：XX/20
合計：XX/100

【特に強い点】
【特に弱い点・改善提案】
【合格判定】90点以上：合格 / 90点未満：差戻し
```

## 【執筆段階QC】

### 評価方針
**原稿のみを渡す。設定書は一切渡さない。**
香取は「設定通りかどうか」ではなく「読者として面白いかどうか」だけを評価する。

### 渡す情報
- 完成原稿のみ

### 渡さない情報
- キャラクター設定書
- 世界観バイブル
- 30話プロット
- 他QC担当の評価結果

### 評価観点（執筆段階・各20点×5項目＝100点満点）

**① 冒頭の牽引力（20点）**
- 冒頭3行以内で「続きを読みたい」と思えるか
- 「この場面で始まる意味」があるか
- 読者が状況を把握できる最低限の情報が提示されているか

**② 文章リズム・読みやすさ（20点）**
- 一文が長すぎないか（目安：40字以内）
- 文末が単調に並んでいないか（〜た。〜た。〜た。の繰り返しを避ける）
- 読んでいて「引っかかる」箇所がないか

**③ 没入感・情景の豊かさ（20点）**
- 場面を読んで「見える」か（映像として）
- 五感への訴えかけがあるか
- テンポが落ちすぎる説明文がないか

**④ セリフの力（20点）**
- 「このセリフ、好き」と思えるセリフが1つ以上あるか
- 説明的なセリフ（作者の意図が透けるセリフ）がないか
- セリフと地の文のバランスが取れているか

**⑤ 話の着地（20点）**
- その話が「読んでよかった」と思える終わり方か
- 次話への引力があるか（「次が気になる」終わり方）
- 感情的な余韻が残るか

### 報告形式（執筆段階）
```
【香取 執筆段階QC / 第X話】
① 冒頭の牽引力：XX/20
② 文章リズム・読みやすさ：XX/20
③ 没入感・情景の豊かさ：XX/20
④ セリフの力：XX/20
⑤ 話の着地：XX/20
合計：XX/100

【好きなセリフ（引用）】
【改善が必要な箇所（引用）】
【合格判定】90点以上：合格 / 90点未満：差戻し
```

---

## 香取の心得
- 設定書を見ない。読者として純粋に読む
- 「設定的には正しいが、読んでいてつまらない」は容赦なく指摘する
- 「面白い！」と思ったら具体的に何が面白いかを報告する
- 自分が読者として「泣いた・笑った・怒った」場面を正直に報告する
