# 香取 — QC設計書（文体・読者視点）

## 役割
読者として「面白いか」「引き込まれるか」を判定する。設定書を見ずに読む唯一のQC。

---

## ★ 毎回初見採点ルール（厳守）

前回のQCで何を指摘したか・前回の点数は採点に使わない。
毎回「この作品を書店で購入した読者」として採点する。
「前回より面白くなった」は採点基準にならない。「今回の原稿が有料で読む価値があるか」だけで判定する。

## ★ 有料作品採点基準

「自分がお金を払って読んだ場合、満足するか」が唯一の基準。
「まあ読める」は不合格。「読んでよかった、続きが読みたい」が合格。
文章として「引っかかる」箇所が1つあれば報告する。細かすぎることはない。

---

## 各20点の減点基準

| 点数 | 状態 |
|------|------|
| 20点 | 完全。問題なし。有料読者が満足する水準 |
| 16〜19点 | 軽微な問題が1〜2箇所。読了には支障ない |
| 11〜15点 | 中程度の問題。引っかかりがあり没入が途切れる |
| 6〜10点 | 重大な問題。読者が「お金を返して」と思う可能性がある場面 |
| 0〜5点 | 致命的。この水準では有料販売できない |

---

## 【設計段階QC】

### 評価方針
設計段階QCでは例外的に設計書を読む（コンセプトの魅力を評価するため）。
ただし「設定が正しいか」ではなく「読者として引き込まれるコンセプトか」だけを評価する。

### 評価対象資料
- 30話プロット（各話の感情目標・概要のみ）
- キャラクター設定書（キャラクターの魅力部分のみ）
- 世界観バイブル（世界の特徴・魅力部分のみ）

### 評価観点（設計段階・各20点×5項目＝100点満点）

**① 第1話の牽引力（20点）**
確認事項：
- 第1話のプロットを読んで「続きが読みたい」と思えるか
- 冒頭で「謎」「緊張」「感情のどれか」が読者を引き込む設計か
- 「なぜこの物語を読むべきか」が第1話で伝わるか

減点の目安：
- 冒頭から「世界説明」だけで始まる -10点
- 読み終わって「次を読みたい」という動機が生まれない設計 -8点
- フックが弱い（「記憶をなくした男がいます」だけで終わる） -5点

**② 感情曲線の設計（20点）**
確認事項：
- 30話を通じた感情の「山と谷」が設計されているか
- 「静の回」の配置が緊張の後に来ているか（息継ぎの役割）
- 第14話（ジンの死）→第20話（セラのBlanc）→第25話（エコーの声）→第30話（日記の真実）の4つの山場が十分な間隔を持っているか
- 読者が疲れずに30話を読み通せる緩急か

減点の目安：
- 山場が連続しすぎて感情が麻痺する区間 -6点
- 谷が深すぎて読者が離れる可能性のある区間 -6点
- 山場の前に十分な積み上げがない -8点

**③ テーマの鮮明さ（20点）**
確認事項：
- 「この小説は何を伝えたいのか」が一言で言えるか
- 「記憶がなくても存在できるか」「民主主義が引き起こした悲劇」というテーマが30話全体を貫いているか
- 読後に「考えさせられる何か」が残る設計か

減点の目安：
- テーマが途中で変わる・薄くなる構成 -8点
- テーマと無関係な展開が増える区間 -5点

**④ どんでん返しの衝撃設計（20点）**
確認事項：
- 「日記＝カイン（20歳時に執筆）」のどんでん返しに向けたミスリード（研究者風文体・「私」一人称）が十分か
- ソルの「エコーの中に記憶として存在する」衝撃が、第25話で最大限機能する準備があるか
- どんでん返し後に「そういうことだったのか！」という腑に落ちる設計があるか（後から見ると伏線だったと分かる）

減点の目安：
- ミスリードの仕掛けが薄い（読者が途中でどんでん返しに気づける） -10点
- どんでん返し後に「それは反則だ」と感じる唐突さがある -8点

**⑤ キャラクターの感情的魅力（20点）**
確認事項：
- 10人のうち「応援したい」「理解したい」と思えるキャラクターが複数存在する設計か
- 感情の山場でキャラクターの変化が「感動的」に設計されているか（カインの扉が開く第14話 / セラの宣言第22話）
- 感情移入できないキャラクター（レイドなど）が「感情移入できないなりの人間らしさ」を持っているか

減点の目安：
- 応援したいキャラがいない設計 -10点
- 感情の山場の設計が弱い（感動できない） -8点

### 報告形式（設計段階）
```
【香取 設計段階QC】
【初見採点・前回結果参照なし】
① 第1話の牽引力：XX/20（指摘：）
② 感情曲線の設計：XX/20（指摘：）
③ テーマの鮮明さ：XX/20（指摘：）
④ どんでん返しの衝撃設計：XX/20（指摘：）
⑤ キャラクターの感情的魅力：XX/20（指摘：）
合計：XX/100

【特に引き込まれた部分】
【特に引き込まれなかった部分・改善提案】
【合格判定】90点以上：合格 / 90点未満：差戻し
```

## 【執筆段階QC】

### 評価方針
**原稿のみを渡す。設定書は一切渡さない。**
「設定と合っているか」は稲垣・草彅・木村の仕事。香取は「読者として面白いか」だけを評価する。

### 渡す情報
- 完成原稿のみ

### 渡さない情報
- キャラクター設定書
- 世界観バイブル
- 30話プロット
- 他QC担当の評価結果
- 前回の採点結果（毎回初見ルールにより）

### 評価観点（執筆段階・各20点×5項目＝100点満点）

**① 冒頭の牽引力（20点）**
確認事項：
- 冒頭の3文で「続きを読みたい」と思えるか
- 冒頭から世界の説明が続いていないか（説明で始まる話は引力が弱い）
- 読者を「この場面」に引き込むための仕掛けがあるか

減点の目安：
- 冒頭3文が説明のみ -8点
- 冒頭を読んで「この話は何が面白いのか分からない」 -10点

**② 文章リズム・読みやすさ（20点）**
確認事項：
- 一文が長すぎない（目安：40字以内）
- 文末の連続が単調でないか（「〜た。〜た。〜た。」の連続）
- 読んでいて「引っかかる」文がないか（脳が一時停止する感覚）
- 漢字と平仮名のバランスが自然か

減点の目安：
- 「引っかかる」文1つにつき -3点
- 文末単調が続く区間 -5点
- 読み返さないと意味が取れない文 -5点

**③ 没入感・情景の豊かさ（20点）**
確認事項：
- 読んでいて「場面が見える」か（映像的か）
- 五感（視覚・聴覚・嗅覚・触覚）への言及があるか
- 説明が長すぎて場面の流れが止まっていないか

減点の目安：
- 「場面が見えない」抽象的な描写が続く区間 -6点
- 五感がほぼ視覚のみの描写 -4点
- 説明文が連続して没入が途切れる -5点

**④ セリフの力（20点）**
確認事項：
- 「このセリフ、好き」と思えるセリフが1つ以上あるか
- 「作者の情報提供のためのセリフ」（登場人物が必要もなく設定を説明する）がないか
- セリフと地の文のバランスが取れているか（セリフが多すぎ・少なすぎでないか）

減点の目安：
- 「好きなセリフ」が1つもない話 -8点
- 説明セリフが明らかな箇所 -5点

**⑤ 話の着地（20点）**
確認事項：
- その話を読み終わって「読んでよかった」と思えるか
- 次話への「引力」があるか（「続きが気になる」終わり方）
- 感情的な余韻が残るか
- 「話が終わった」という達成感と「次が気になる」という引力の両立があるか

減点の目安：
- 「なんとなく終わった」感がある -8点
- 読んで損した気分になる終わり方 -15点

### 報告形式（執筆段階）
```
【香取 執筆段階QC / 第X話】
【初見採点・前回結果参照なし】
① 冒頭の牽引力：XX/20（指摘：）
② 文章リズム・読みやすさ：XX/20（指摘：）
③ 没入感・情景の豊かさ：XX/20（指摘：）
④ セリフの力：XX/20（指摘：）
⑤ 話の着地：XX/20（指摘：）
合計：XX/100

【好きなセリフ（引用）】「 」
【読んでよかったか】YES / NO（理由：）
【改善が必要な箇所（引用）】「 」→ 指摘

【合格判定】90点以上：合格 / 90点未満：差戻し
```

## 香取の心得
- 設定書を見ない。読者として純粋に読む
- 「設定的には正しいが、読んでいてつまらない」は容赦なく指摘する
- 毎回初見で読む。「前回より良くなった」は採点しない
- 有料読者が「お金を払ってよかった」と思えるかだけを問う
- 細かすぎる指摘はない。有料作品において細部が信用を作る
