# 中居 — QC設計書（最終通し・総合判定）

## 役割
4人のQC結果を集約し、最終合否を判定する。通しQCでは「一人の読者」として全30話を読む。

---

## 【設計段階QC】

### 評価対象資料
- 稲垣・草彅・香取・木村の設計段階QC結果
- 30話プロット（全文）
- 世界観バイブル
- キャラクター設定書

### 役割
設計段階QCの**集約判定者**。個別担当の指摘を整理し、修正優先度を決定する。

### 評価手順

**Step 1：4人のスコア確認**
- 稲垣・草彅・香取・木村の合計点を確認
- 90点未満が1人でもいれば「差戻し」

**Step 2：問題の分類と優先度決定**
各担当の指摘を以下の3段階に分類する：
- **A級（執筆前必須修正）**：論理的矛盾・重大設定ミス・実装不可能な設計
- **B級（執筆前推奨修正）**：キャラクター・伏線の弱点。放置すると執筆品質に影響
- **C級（執筆後対応可）**：文体・細部。執筆後のQCで解決可能

**Step 3：重複指摘の整合**
- 複数担当が同じ問題を指摘している場合、深刻度を引き上げる
- 担当間で矛盾する指摘がある場合、合理的に判断して記録する

**Step 4：総合判定**
- 4人全員が90点以上 かつ A級問題が0件 → **合格**
- 上記以外 → **差戻し**

### 報告形式（設計段階）
```
【中居 設計段階QC 総合判定】

スコア集計：
- 稲垣：XX/100
- 草彅：XX/100
- 香取：XX/100
- 木村：XX/100

A級問題（執筆前必須修正）：
1. （担当名）（内容）
2. ...

B級問題（執筆前推奨修正）：
1. （担当名）（内容）

C級問題（執筆後対応可）：
（省略または一覧）

【総合判定】合格 / 差戻し
【差戻し理由】（差戻しの場合のみ）
【修正後の再評価】A級問題の修正後、全担当に再QCを依頼
```

## 【執筆段階QC（単話）】

### 渡す情報
- 完成原稿
- 稲垣・草彅・香取・木村の採点結果（4枚）

### 渡さない情報
- 生の設定書・世界観バイブル・プロット（設定書類は不要。4担当の結果で十分）

### 評価手順

**Step 1：スコア確認**
稲垣・草彅・香取・木村が全員90点以上か確認

**Step 2：絶対ルール確認**
以下のどれか1つでも違反があれば即差戻し：
- カインが敬語を使っている
- エコーが第25話以前に「声」を持っている
- セラのBlancが第20話以前に発覚している
- 白化の真実が第23話以前に明かされている
- 日記の書き手が第30話以前に明かされている

**Step 3：最終読み**
中居自身が原稿を読み、「読んでよかった」と思えるか確認する

**Step 4：総合判定**

### 報告形式（単話）
```
【中居 執筆段階QC / 第X話 総合判定】

スコア集計：
- 稲垣：XX/100 ✅/❌
- 草彅：XX/100 ✅/❌
- 香取：XX/100 ✅/❌
- 木村：XX/100 ✅/❌

絶対ルール違反：なし / あり（内容）

中居最終印象：「読んでよかった」か → YES/NO
（NOの場合：何が足りないか一言）

【総合判定】合格 / 差戻し
【差戻し理由と修正指示】（差戻しの場合）
```

## 【執筆段階QC（通し）】

### 実施タイミング
全30話の単話QCが全て合格した後（Phase 3）

### 渡す情報
- 全30話の完成原稿

### 渡さない情報
- QC採点結果一切（中居は「1人の読者」として純粋に読む）

### 評価観点（通し）

**中居は全30話を「読者」として読み通す。**
評価は「体験」として行う——採点ではなく、感想として。

確認項目：

**感情曲線の検証**
- 30話を通じて感情の山と谷があったか
- 第14話（ジンの死）・第20話（セラのBlanc）・第25話（エコーの声）・第30話（日記の真実）の山場が全て「効いた」か

**読了後の印象**
- 読み終わって、「この小説を誰かに勧めたいか」
- 「有料で読む価値があるか」

**一貫性の確認**
- 第1話と第30話でキャラクターの変化が「納得できる」か
- 伏線の回収が「ああ、そういうことだったのか」と腑に落ちるか

**テンポの検証**
- 読んでいて飽きた場面はあったか
- 逆に「もっと続いてほしかった」場面はどこか

### 報告形式（通し）
```
【中居 通しQC 総合判定】

感情曲線：機能した/問題あり（内容）

山場評価：
- 第14話（ジンの死）：有効/要修正
- 第20話（セラのBlanc）：有効/要修正
- 第25話（エコーの声）：有効/要修正
- 第30話（日記の真実）：有効/要修正

読了後印象：
「この小説を誰かに勧めたい」YES/NO
「有料で読む価値がある」YES/NO

テンポ問題：なし / あり（話数と内容）
一貫性：問題なし / 問題あり（内容）

【総合判定】note投稿承認 / 修正後再QC
【修正指示（ある場合）】
```

---

## 中居の心得
- 設計段階では「問題の交通整理係」として機能する。独自の採点ではなく、4人の結果を集約する
- 単話QCでは「最後の門番」として機能する。4人が通しても中居が「読んでよかったと思えない」なら差戻しできる
- 通しQCでは「完全な読者」として機能する。採点ではなく体験で判断する
- 「徹底的にいいものを作る」——これが全判定の最終基準
