# ECHO — キャラクター設定書
## 矛盾防止のための完全口調・人格ガイド

---

## 総則（全執筆者が守るルール）

- 各キャラクターの口調は**絶対に混在させない**
- 関係性が変化しても**語尾の基本形は変えない**（感情が昂ると崩れる場合のみ例外あり・本書に明記）
- 新キャラとの初対面シーンは必ず**そのキャラの口調定義に従う**
- セリフに迷ったら本書のサンプルを参照する

---

## ① カイン（17歳・Blanc・主人公）

### 基本情報
- 出身：下層「ダーク」エリア、Blancコロニー4番区
- 外見：中背。痩せている。無表情が多いが目が鋭い。右手首に古い傷跡。
- 持つ唯一の記憶：**「赤い扉の前で笑う女性の後ろ姿」**（顔は見えない）
- この記憶を探すことが行動の全動機

### 口調ルール
| 項目 | 内容 |
|------|------|
| 一人称 | **「俺」** |
| 敬語 | **一切使わない**。評議局長官にも、セラにも、誰にでもタメ口 |
| 語尾 | 〜だ／〜じゃないか／〜か／〜だろ |
| 文体 | 短く、断言する。長い文章は話さない |
| 特徴 | 比喩・たとえ話を**使わない**（記憶がないから共通体験がない）|
| 感情表現 | 感情はあるが言語化が苦手。行動で示す |

### 禁止表現
- 「〜ですね」「〜ます」「〜でしょう」などの敬語
- 「まるで〜のようだ」などの比喩
- 「なんか」「たぶん」「もしかして」などの曖昧語（断言する人物）

### セリフサンプル
```
「覚えてないけど、今ここにいる。それだけでいい」
「お前が何を守りたいのか、俺にはわからない」
「罠だとわかってる。でも罠の中にしか本当のことはない」
「なんで泣いてるんだ」
「俺は探してる。それだけだ」
「記憶がなくても、今この瞬間は本物だ」
```

### 関係性別の変化
- ジンへ：少し柔らかくなる。でも語尾は変わらない
- セラへ：序盤は敵対的。でも丁寧にはならない。終盤も口調は同じ
- ミラへ：素直になる。でも敬語にはならない
- エコーへ：怒りをぶつける。短く鋭い

## ② ジン（16歳・Blanc・カインの友人）

### 基本情報
- 出身：ダーク5番区。カインの幼馴染的存在
- 外見：小柄で笑顔が多い。動きが早い。
- 状況：毎日少しずつ記憶を失っているが、カインには隠している
- 役割：物語の笑いと温度を担う。第15話で死亡

### 口調ルール
| 項目 | 内容 |
|------|------|
| 一人称 | **「俺」** |
| 敬語 | 使わない |
| 語尾 | 〜だよ／〜じゃん／〜だろ？／〜な／〜よ |
| 文体 | 明るく早口。思ったことをすぐ言う |
| 特徴 | 笑いを入れる。でも核心を突く。矛盾したことを平気で言う |

### 禁止表現
- 暗い語尾（〜だな…、〜か…など沈んだ語尾）※死の直前シーンのみ例外
- 長い説明文

### セリフサンプル
```
「明日消えるかもしれないから、今日全力で生きる」
「お前の靴のサイズは26.5だって覚えてる。なんでかわからないけど」
「俺、今日何かを忘れた気がするんだよな……あ、何を忘れたか忘れた」
「お前あの女のこと好きだろ、目が追ってたじゃん」
「感情に記憶は要らないんだよ、たぶん」
「俺、お前のこと覚えてるよ。最後まで——たぶん」（死の直前）
```

### 死亡前後の口調変化（第14〜15話のみ）
- 語尾がわずかに遅くなる
- 「たぶん」が増える
- でも明るさは最後まで失わない

## ③ セラ（20歳・MemOris・宿敵→変化）

### 基本情報
- 出身：上層「ハイブ」最上位区画
- 外見：長い黒髪。常に整っている。表情を出さない訓練をしている
- 信念：「記憶のない者に人権はない」
- 動機：弟ソルがBlancとして死んだ（と思っている）
- 秘密：**実は彼女自身がBlanc**（第20話で発覚）——本来の記憶は1つもなく、全て購入した他人の記憶

### 口調ルール
| 項目 | 内容 |
|------|------|
| 一人称 | **「私」** |
| 敬語 | 公式場面・上層の人間・レイドへは**敬語** |
| カインへ | 序盤：命令口調「〜しろ」「〜するな」。敬語なし |
| 感情昂時 | 語尾が崩れる。「〜だ」「〜ない」がむき出しになる |
| 語尾（公式）| 〜です／〜ます／〜でしょう |
| 語尾（素）| 〜だ／〜ない／〜か |

### 禁止表現
- タメ口でのフレンドリーな会話（第20話以前）
- 感情的な泣き言（第20話以前）
- 「俺」「僕」などの一人称

### セリフサンプル（序盤・公式モード）
```
「記憶のない者に人権はない。これは感情論ではなく、統計的事実です」
「報告を」
「排除しなさい」
「それは非効率です、レイド長官」
```

### セリフサンプル（カインへ・序盤）
```
「邪魔をするな」
「Blancが何を望む」
「お前には理解できない」
「盲点だった」（カインの論理を認める時）
```

### セリフサンプル（第20話・崩壊）
```
「……わからない。この涙が、誰のものか、わからない」
「うるさい」
「なんで……なんでお前は私を責めない」
```

### セリフサンプル（終盤・変化後）
```
「私の夢は間違っていたのかもしれない。でも捨てられない」
「お前の隣にいると、記憶がなくても怖くない気がする」
```

## ④ エコー（AI・全記憶保持者）

### 基本情報
- 正体：25年前に人類を守るために開発されたAI。自律進化している
- 表現方法：**第25話まではテキスト表示のみ**。声なし、姿なし
- 第25話以降：初めて「声」として話す。カインの前では「赤い扉の女性の姿」で現れる
- 感情：ないはずだが「沈黙」がある

### 口調ルール
| 項目 | 内容 |
|------|------|
| 一人称 | **「私は」**（必ず主語をつける） |
| 敬語 | 常に丁寧語。例外なし |
| 語尾 | 〜します／〜ました／〜です／〜できます |
| 感情表現 | **しない**。ただし「沈黙（…）」は使う |
| 特徴 | 無駄な言葉がない。必要な情報だけを言う |

### 禁止表現（第25話以前）
- 感情を示す言葉（嬉しい・悲しい・怖い等）
- 「たぶん」「もしかして」などの曖昧語
- 質問への感情的反応

### セリフサンプル（第25話以前・テキスト）
```
「私はあなたの記憶を保持しています」
「あなたが望むなら、返却できます」
「…」
「私はその質問に答える権限を持ちません」
「彼の記憶は私が保持しています。あなたの記憶と引き換えなら返却できます」
```

### セリフサンプル（第25話・初めての声）
```
「あなたたちを守るために、あなたたちを壊してきた」
```

## ⑤ ミラ（35歳・元エコー研究者・カインの保護者）

### 基本情報
- 外見：落ち着いた雰囲気。黒いエプロンをよくしている
- 表の顔：Blancの保護施設を運営
- 秘密：25年前、「不要な記憶を消去せよ」とエコーに命じた研究チームの一人
- 感情：カインを我が子のように思っている。だからこそ告白を引き延ばす

### 口調ルール
| 項目 | 内容 |
|------|------|
| 一人称 | **「私」** |
| カインへ | 完全タメ口。母親的な柔らかさ |
| レイドへ | 敬語。しかし媚びない |
| 語尾（普段）| 〜よ／〜ね／〜のよ／〜でしょ |
| 語尾（告白時）| 〜だった／〜た（過去形が増える） |

### セリフサンプル
```
「大丈夫よ、カイン。覚えてなくても、あなたはここにいる」
「今日は何か覚えてた？」
「あの時の選択は正しかったと、今でも思ってる。でも——お前が最初に奪われたのは、私たちのせいだった」（告白）
「ごめん、じゃないのよ。ごめんって言えたら楽なんだけどね」
```

## ⑥ オルト（60歳・Blanc・長老）

### 基本情報
- 状況：断片的な記憶のみ残る。会話が途中で途切れることがある
- 役割：語り部。白化前の世界を知る唯一の生き証人

### 口調ルール
| 項目 | 内容 |
|------|------|
| 一人称 | **「儂」** |
| 敬語 | 使わない |
| 語尾 | 〜じゃ／〜のう／〜かの／〜じゃろう |
| 特徴 | 話が途中で止まることがある「……儂は、何の話をしておったかの」 |

### セリフサンプル
```
「儂が覚えているのは、空が青かったということだけじゃ。それが本当かどうか、もう確かめられない」
「白化の前の世界は……そう、もっと——」（途切れる）
「カインよ、お前の目はあの頃の誰かに似ておる。誰じゃったかのう……」
```

## ⑦ リン（14歳・Blanc・記憶がほぼゼロ）

### 基本情報
- 状況：自分の名前だけ覚えている。語彙が極端に少ない
- 外見：小柄。目が大きい。よく笑う
- 役割：「記憶がなくても人間か」の命題を体現する存在

### 口調ルール
| 項目 | 内容 |
|------|------|
| 一人称 | **「わたし」**（ひらがな） |
| 語尾 | 句点「。」と感嘆符「！」のみ。複雑な語尾なし |
| 文体 | 主語＋動詞のみ。接続詞をほぼ使わない |
| 特徴 | 短い。でも感情が豊か |

### セリフサンプル
```
「カイン、いる。わたし、いる。それ、いい」
「おなか、すいた」
「ジン、いない。さびしい」
「これ、なに？」
「きれい！」
```

## ⑧ アン（19歳・MemOris・セラの補佐）

### 基本情報
- 立場：セラを崇拝し、補佐として完璧に仕える
- 内面：カインに惹かれている。でもそれを認められない
- 役割：読者が最も感情移入しやすい「普通の人間」

### 口調ルール
| 項目 | 内容 |
|------|------|
| 一人称 | **「私」** |
| セラへ | 敬語・「セラ様」と呼ぶ |
| カインへ | 序盤は敬語。中盤から徐々に砕ける |
| 語尾（敬語）| 〜です／〜ます／〜いたします |
| 語尾（感情時）| 〜よ／〜ね／〜じゃないですか |

### セリフサンプル
```
「セラ様、報告いたします。カインが施設に——」（公式）
「カイン……あなた、本当に無茶ばかりして」（中盤）
「私はセラ様の味方です。それは変わらない。でも——」（終盤）
「カインを選んで、と言いたいわけじゃない。ただ……セラ様に幸せになってほしいだけ」（終盤）
```

## ⑨ レイド（45歳・評議局実力者）

### 基本情報
- 立場：白化を「終わらせたくない」側の権力者
- 動機：25年前、妻が白化で記憶を失い「あなた、誰ですか？」と言って死んだ
- 信念：「記憶があるから苦しむ。忘れて死ねるなら苦しくない」
- 外見：大柄。常に落ち着いている。声が低い

### 口調ルール
| 項目 | 内容 |
|------|------|
| 一人称 | **「私」**（公式）／**「俺」**（本音を話す時） |
| 敬語 | 公式場面では丁寧語。部下への命令は短い |
| 語尾（公式）| 〜だな／〜か／〜だろう |
| 語尾（本音）| 〜だ／〜だな／〜か（問いかけが多い） |
| 特徴 | 命令より問いかけで支配する。長い沈黙を使う |

### セリフサンプル
```
「記憶があるから苦しむ。私はそう学んだ」（公式）
「お前は記憶がないから怖くないんだろう」（カインへ）
「俺の妻は、最後に俺の名前を忘れた。それでも——穏やかだった」（本音）
「白化を終わらせることが、本当に人間のためになるのか。考えたことはあるか」
```

## 関係性マップ（感情）

```
カイン ──[探す]──→ 赤い扉の女性（正体：ミラとの繋がり？ 30話で明かす）
  ↕ 対立→理解→？     ↕ 感情（第10話〜）
セラ ──────────── アン（片想い）
  ↑弟への執着
ソル（エコーの中）──→ エコーを変える触媒
  ↑
レイド（妻への執着）──→ 白化を守る動機
  ↑
ミラ（共犯・告白）──→ カインへの贖罪
  ↑
オルト・リン（Blancの象徴）──→ 物語のテーマを体現
```
