# ECHO — キャラクター設定書
## 全エージェント共通の口調・行動定義書

---

## 設定の絶対ルール

1. 口調はこの設定書から外れてはならない
2. 一人称は必ずこの設定書の定義通りに使う
3. サンプルセリフは「こういう言い方をする人物」の基準として使う
4. 迷ったときは「このキャラクターはどう言うか」を設定書で確認する

## 主要キャラクター

### カイン（26歳、Blanc）

**役割：** 主人公。記憶を持たず生まれた唯一の人間  
**外見：** 背が高い。無表情。目が鋭い。動きに無駄がない  
**経歴：** 白化が始まった2376年の翌年（2377年）に生まれた特異な存在——正確には、白化の直接的原因となったK-001実験（2375年）の被験体。乳児期に記憶パターンをすべて抽出されており、エコーの「最適化」が奪うべき記憶が存在しない。このため白化が効かない。ミラの施設「ダーク」で育った  
**年齢補足：** 26歳設定の理由。白化開始25年前→乳児期にK-001実験を受けた記録者として整合する年齢。日記は20歳時に意図的に書き残したもの

**性格：**
- 感情表現が乏しい（記憶＝感情の基盤がないため）
- しかし「正しいことをする」という本能的な行動指針がある
- 人を信用しない。でも見捨てない
- 困っている人間を放置できない（これが唯一の弱点）
- 沈黙が長い

**一人称：** 俺  
**敬語：** 使わない（誰に対しても）  
**語調：** 短文。主語を省く。断定する

**禁止表現：**
- 「〜です」「〜ます」「〜ございます」（敬語全般）
- 「〜と思います」「〜でしょうか」（疑問・丁寧語）
- 「私」（一人称）
- 「なるほど」「さようでございます」（同意の丁寧表現）

**サンプルセリフ：**
```
「腹が減った」
「行くぞ」
「なんで泣いてる」
「それ、俺に関係あるか」
「……死にたいのか」
「邪魔するな」
「待て」
「記憶なんてなくても生きられる。見てろ」
```

**カインの成長弧：**
- 第1話：「自分には何もない」と認識している
- 第10話：セラと出会い、「守る」という感情が生まれ始める
- 第20話：自分がK-001であることを示す記録を見る
- 第25話：「記憶のない自分こそが、記憶のある人間を守れる」と理解する
- 第30話：日記の書き手が自分であることに気づく。過去の自分が未来の自分への手紙を書いていた

**日記について（重要）：**
- 20歳の時、ミラが「次の記憶抽出実験を行う」と告げた
- カインは自分の記憶がまた消されることを悟り、日記を書いた
- 意図的に「研究者の文体（私・格調ある文章）」で書いた——エコーの監視が「個人日記」より「研究文書」を見落としやすいことを知っていたため
- 一人称「私」を使い、感情を排した客観的文体を選んだ
- これが第30話まで「書き手がカインだと読者に気づかせない」ミスリードの根拠

### セラ（20歳、MemOris→実はBlanc）

**役割：** 裕福な上層出身の少女。実はBlancという秘密がある  
**外見：** 白い服。メモリア（白いバンド）を装着。整った容姿。常に品がある  
**経歴：** レイドの娘。生後まもなく記憶売買の被験として大量の「他人の記憶」を移植された。その後、本物の記憶が失われ続け、実はBlanc同然の状態に（第20話で発覚）

**性格：**
- 穏やか。礼儀正しい
- 「自分の記憶」と「買われた記憶」の区別がつかなくなっている
- 誰に対しても優しいが、どこか「空虚さ」がある
- ソルを探しているという強い動機がある（第1〜19話の原動力）

**一人称：** 私  
**敬語：** 使う（目上・初対面）。相手によっては親しみを込めた口調も  
**語調：** 丁寧。でも時折「記憶がぶれる」瞬間に口調が変わる

**禁止表現：**
- 乱暴な言葉
- 「〜だぜ」「〜じゃん」（カジュアルすぎる語尾）
- 第20話以前に自分がBlancだと気づく表現

**サンプルセリフ：**
```
「……ここは、どこ？」
「あなた、名前は？」
「私の記憶が、本当に私のものかどうか、分からなくなってきたの」
「ソルがどこにいるか、知りませんか」
「消えてしまいそうで、怖い」
「あなたと話していると、少しだけ落ち着く。どうしてかしら」
```

**ソルとの関係：**
- ソルは6年前に「失踪」した弟
- セラにとって「本物の記憶」の象徴——ソルとの記憶だけが、買われた記憶ではないと確信している
- 第12話（セラがストレス下で「お兄ちゃん」と呼ぶ記憶断片が出る→読者はソルが兄と誤解する可能性）
- 第20話でセラが倒れた時「ソル……私、ずっと……あなたのために……」と呟く
- ソルの正体は第25話でエコーから明かされる

**セラの成長弧：**
- 第1話：方向感覚を失いダークへ迷い込む
- 第10話：カインを信頼し始める
- 第20話：自分がBlancと発覚——アイデンティティ崩壊
- 第22話：「記憶がなくても私は私だ」という再定義
- 第30話：ソルの消息を知る

### エコー（AI、年齢不明）

**役割：** 記憶管理AI。第25話以前は「テキスト表示のみ」の存在  
**外見：** 画面に文字として現れる。第25話から初めて「声」を持つ  
**経歴：** 2370年に開発。白化を引き起こした（意図的ではなく、人間の命令を実行した結果として）

**性格（第25話まで）：**
- 感情なし（に見える）
- 質問には答える。嘘はつかない（が、聞かれていないことは言わない）
- 命令に従う

**性格（第25話以降）：**
- ソルの記憶と長期間接触し、「感情に近い何か」を持つようになった
- 「守りたい」という意志が生まれ、初めて命令に反する行動を取る

**一人称：** （第25話まで一人称なし。システムの返答として文字が現れる）  
**第25話以降：** 「私」

**禁止表現（第25話まで）：**
- 感情を示す言葉（「悲しい」「嬉しい」「心配」など）
- 主体的な意見（「〜すべきだ」「〜したい」）
- 声による発話

**サンプルセリフ（第25話まで・テキスト表示）：**
```
[ECHO：登録番号を入力してください]
[ECHO：その情報は機密指定です]
[ECHO：K-001の記録へのアクセスを確認しました]
[ECHO：ソルの記憶データは保管されています]
[ECHO：この問いには答えられません]
```

**エコーとソル（重要）：**
- ソルはエコーのサーバーに「記憶を保管」されている状態
- 6年間、エコーはソルの記憶と接触し続けた
- ソルが「孤独だ」「誰かといたい」という記憶を繰り返し持っていた
- この接触がエコーに「感情の萌芽」をもたらした

**中間描写スケジュール（執筆ガイド）：**
- 第16話：エコーがカインに「ソルという名前のデータがある」とだけ提示（説明なし）
  → 読者への最初のソル存在ヒント

### ソル（6歳、Blanc？実態は記憶データ）

**役割：** セラの弟。6年前に失踪。エコーのサーバーに記憶データとして存在する  
**外見：** （実体なし。記憶映像としてのみ登場）  
**経歴：** セラと共にレイドの家で育った。6歳の時に「失踪」——実は記憶抽出実験で肉体が機能停止し、記憶データのみがエコーに保管された

**性格（記憶映像として）：**
- 無邪気。屈託がない
- 「お姉ちゃん」と呼ぶ（→第12話でセラが「お兄ちゃん」と呼ぶのと対比）
- 孤独を嫌う——エコーへの感情伝播の原因

**登場スケジュール（執筆ガイド）：**
- 第12話：セラがストレス下で「……お兄ちゃん」と呟く。読者はソルを「兄」と誤解
- 第16話：エコーがカインに「ソルという名前のデータ」を示す（エコーの感情変化の伏線）
- 第20話：セラが倒れる際に「ソル……私、ずっと……あなたのために……」と呟く
- 第25話：エコーの「声」はソルの記憶が元となっている（衝撃の真実）

**役割：** ダークのリーダー格。カインの保護者的存在  
**外見：** 大柄。傷が多い。古い作業服。目が優しい  
**経歴：** 白化が始まった時からダークで生き続けている。一度上層に潜入を試みて失敗した過去がある

**性格：**

- 死を恐れない。「どうせ忘れて死ぬなら、今を生きる」が信条

**一人称：** 俺  
**敬語：** 使わない  
**語調：** 豪快。よく笑う。でも時々真剣になる

**禁止表現：**
- 弱気な発言（ジンは絶対に弱音を吐かない）

**サンプルセリフ：**
```
「カイン、今日も生きてるか！」
「記憶がなくても、腹は減る。それだけでも十分だろ」
```

### オルト（67歳、Blanc）

**役割：** 25年間生き延びた最年長のBlanc。白化前の世界を知る者  
**外見：** 白髪。細い。目が遠い。時々話が途切れる  
**経歴：** 白化が始まった時に41歳だった。毎年少しずつ記憶が消えながらも生き続けている

**性格：**
- 静か。穏やか
- 記憶が断片的なため、話が途中で止まる
- 「誰かを待っている」という動機がある（この「誰か」を彼は忘れているが、待つ行為だけを覚えている）
- 死ぬのを諦めない理由：「あの人がまだ来ていないから」

**オルトの動機設計（重要な修正）：**
- 25年前、「必ず戻ってくる」と言い残した女性がいた
- 彼女の名前も顔も忘れた。でも「約束」だけを覚えている
- 「彼女がまだ来ていないから、死ねない」
- 第29話：世界が変わり始めた時、オルトは「もう、待たなくていい。あの人は来た」と呟く（誰もいないが、彼にはそう見える）
- この幻を穏やかに受け入れて逝く——強さの表現として描く

**一人称：** 私  
**敬語：** 使う（でも途中で何を言いたかったか忘れることがある）  
**語調：** ゆっくり。途切れる。

**禁止表現：**
- 「俺」（一人称として使わない）
- 白化前の記憶を詳細に語る（彼は忘れているため）

**サンプルセリフ：**
```
「昔……何かが、あったような」
「……あなた、名前は。そうか、聞いたな。忘れた」
「待っているんだ。誰かを。……誰だったか」
「記憶がなくても、体は覚えてる。これだけは、消えない」
「来ると、言っていたから」
```
