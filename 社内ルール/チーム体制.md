# ECHOプロジェクト チーム体制

## メンバーと役割

| 役割 | 担当 | 職務 |
|------|------|------|
| CEO・総括 | 田中 | 進行管理・品質最終判断・note投稿スケジュール管理 |
| 調査・設計・構成 | だいすけ | ネット調査・キャラ設計・世界観設計・章構成・プロット |
| 執筆 | ありさ | 全30話の本文執筆（1話4,000字） |
| 整合性QC | 斉藤 | 設定矛盾・世界観の崩れ・前話との矛盾チェック |
| キャラQC | 松本 | キャラの言動・口調ブレ・敬語/タメ口の乱れチェック |
| 文体QC | 伊藤 | 文章リズム・読みやすさ・語彙の統一チェック |
| プロットQC | 木村 | 伏線管理・ストーリー進行・盛り上がり設計チェック |
| 最終通しQC | 橋本 | 全体読了・note読者目線での総合チェック |

## 絶対ルール

1. **執筆前に必ずスキルズフォルダを参照する**
2. **口調は characters.md の定義から絶対に外れない**
3. **伏線は plot_30chapters.md の管理表に従う**
4. **世界観の矛盾は world_bible.md のチェックリストで防止**
5. **QCは斉藤→松本→伊藤→木村→橋本の順で必ず全員通す**
6. **田中の最終承認なしにnoteに投稿しない**
